# Hyperliquid API Documentation

## Overview
Documentation for the Hyperliquid public API.

**Official Python SDK:** [https://github.com/hyperliquid-dex/hyperliquid-python-sdk](https://github.com/hyperliquid-dex/hyperliquid-python-sdk)

### Community SDKs
* **Rust SDK** (by Infinite Field): [https://github.com/infinitefield/hypersdk](https://github.com/infinitefield/hypersdk)
* **TypeScript SDKs**: 
    * [https://github.com/nktkas/hyperliquid](https://github.com/nktkas/hyperliquid)
    * [https://github.com/nomeida/hyperliquid](https://github.com/nomeida/hyperliquid)
* **CCXT**: [https://docs.ccxt.com/#/exchanges/hyperliquid](https://docs.ccxt.com/#/exchanges/hyperliquid)

### API Endpoints
* **Mainnet URL**: `https://api.hyperliquid.xyz`
* **Testnet URL**: `https://api.hyperliquid-testnet.xyz`

### Important Sub-sections (Available on GitBook)
The following detailed information is available in the sidebar of the official documentation:
* **Notation**: Mathematical and type definitions.
* **Asset IDs**: Unique identifiers for assets.
* **Tick and lot size**: Trading constraints.
* **Info endpoint**: Public data queries (L2 book, metadata, positions, etc.).
* **Exchange endpoint**: Trading actions (Place order, cancel, etc.).
* **Websocket**: Real-time data streaming.
* **Signing**: Authentication and request signing implementation.
* **Rate limits**: API usage constraints.

Full Documentation: [https://hyperliquid.gitbook.io/hyperliquid-docs/for-developers/api](https://hyperliquid.gitbook.io/hyperliquid-docs/for-developers/api)
